# Linux常用命令

### 1.系统路径：

```bash
/	表示根目录
~	表示/root
etc	存放系统配置目录
home	除了root以外所有用户默认在home下新建一个以用户名作为文件夹名称的文件夹
root	用户root单独文件夹
usr	所有用户安装的软件都放入到这个文件夹中
```
### 2.常用命令：

```bash
$ pwd	打印工作目录，即当前文件夹全路径

$ su username 切换当前用户

$ cd	进入到目录

$ cd ..	返回上一级目录

$ mkdir [-p]	新建目录

$ ls	平铺

$ ll	详细列表

$ vi和$ vim	编辑文件，如果文件不存在则创建文件。进入编辑器后，默认只读状态，点击"i"或者"insert"启用编辑状态，编辑完成后点击"esc"退出编辑状态
	:Wq	保存并退出
	:q	不保存退出
	:q!	强制退出

$ touch	创建空文件

$ cat	查看文件全部内容

$ head [-n] 文件名	查看文件前n行，默认前10行

$ tail [-n] 文件名	查看文件后n行，默认后10行
	$ tailf	动态显示文件后n行，常用在显示tomcat日志文件功能

$ echo '内容' >> 文件名	向文件中添加一些内容

$ ifconfig	打印网卡信息

$ reboot	重启

$ tar zxvf 文件名	解压文件

$ cp [-r](复制文件夹时需要-r参数) 原文件 新文件路径	复制文件

ctrl+C	中断

$ mv 原文件 新文件 剪切(具备重名名功能)

$ rm 删除	加[-f]表示强制删除，不用问；加[-r]表示递归
```
